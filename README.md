NEU Library Visitor Log System
https://polite-chebakia-9896a9.netlify.app/

Description
This project is a web-based NEU Library Log System that allows users to log their visit information and enables an administrator to manage and monitor visitor data. The system uses Google authentication to log in users and determines whether the user is a regular user or an admin based on their role in the database. Regular users can input their visit details, while the admin has access to additional features for viewing and managing visitor records. The project demonstrates the use of role-based access control and basic data management in a simple web application.
